# A2_iOS_Hossein_101508476

iPhone app built with **SwiftUI** and **Core Data** for browsing and adding products. Course lab assignment (Product App).

## Features

- **Home:** shows the current product; **Previous** / **Next** move through products in order  
- **All Products:** full list of saved products  
- **Search:** filter by product **name** or **description**  
- **Add Product:** form to create a new product (validated before save)  
- Sample products are seeded when the store is empty (for a quick start)

## Tech stack

- SwiftUI  
- Core Data  
- Xcode (iOS)

## Build and run

Building and running this project requires **Xcode on macOS** (not verifiable from Windows alone).

1. Open **`A2_iOS_Hossein_101508476.xcodeproj`** in Xcode.  
2. Select an **iPhone Simulator** (or a device).  
3. Run (**⌘R**).
